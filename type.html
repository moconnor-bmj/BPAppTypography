Notes on these pages

Body Text: This is an attempt to help the App Developers create a typographical vertical rhythm for the BP Window App. It should improve the aesthethics and readability of the content. It works by setting the base font and line-height (16px & 1.5) and all other font sizes are made relative to these. If we change the base size, everything will change proportionally...

As we don't have an emulator to test the settings in context, this is a stab in the dark to some extent. If it does look ridiculous when implemented, tweaking the base font should allow for easy modification. Declaring an HTML font-size of 62.5% (see vr.css) makes 1em equal to 10px and so forth. Thus 1.6em = 16px. To increase or decrease font size adjust the HTML % ...

I'm not sure what the coventions are for Windows App Dev and this may not work, but I think its a good guide to work from if thats the case.

I'm working from a blank canvas - that may not be possible in the case of BP App code ;-)

I have deliberately not mentioned font faces / types as this is type agnostic!!
